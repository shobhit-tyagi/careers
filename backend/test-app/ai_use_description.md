# AI Tool Usage Disclosure

This document describes how AI tools were used during the assessment.

## Tools Used
- ChatGPT
- GitHub Copilot

## How They Were Used
- I am not a Node.js developer, so I used AI tools to assist with code generation at a granular level.
- I provided the business logic and system intent in natural language, and used AI to translate that into working code.
- Code was generated in small pieces rather than end-to-end generation to ensure correctness and maintain control over structure and integration.
- I manually reviewed and integrated all generated snippets to ensure they fit together correctly and aligned with the intended architecture.

## Testing Approach
- AI was used to help generate test code.
- All generated tests were manually reviewed.
- Special attention was given to ensure that critical and core business logic is properly covered.
- I verified that tests reflect intended behavior rather than blindly accepting generated assertions.

## Summary
AI tools were used as a productivity aid and syntax/code generation assistant. All architectural decisions, integration, and validation of correctness were done manually.
